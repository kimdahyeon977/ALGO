문제 흐름
1. 배추지렁이는 + 방향으로 퍼짐
2. 최소로 필요한 지렁이를 알고 싶음

풀이
- 모든 땅을 한번씩 방문
- 방문된 땅에 대해서 -> 배추 여부 판단
-- 배추일 경우: BFS 사용하여 인접한 땅 방문처리
-- 땅일 경우: 방문처리
- 땅을 한번씩만 방문하게 되므로 O(n*m) 시간 복잡도


함수화
- 기존에 풀어봤던 문제
- 함수화 하면 단계별로 볼 수 있을 것.
적용
장점
- 행위가 잘 보여 코드 읽기가 조금 수월해짐
- 단계별로 생각하기 쉬움
단점
- 코드양 자체가 많아짐
- 단순한 코드일 경우 오히려 코드를 더 작성해야 함


파이썬
가변(mutable) 객체
- 값을 변경할 수 있는 객체 
- 한 번 생성되면 내부의 상태를 변경할 수 있음
- 리스트, 딕셔너리, 집합, 클래스 인스턴스 등

불변(immutable) 객체
- 값을 변경할 수 없는 객체
- 값을 변경하려면 새로운 객체를 생성
- 정수, 실수, 문자열 등

def cabbage(m, n, board, visited):
    count = 0

    # 땅을 한번씩 방문
    for i in range(n):
        for j in range(m):
            # 땅, 배추 판단 -> 방문
            # is_cabbage()
            if is_cabbage(i, j, board, visited):
                count += 1
                bfs(i, j, m, n, visited, board)
            else:
                visited[i][j] = 1
                    
    print(count)


def is_cabbage(x, y, board, visited):
    # 벌레가 없는 배추일 경우
    if not visited[x][y] and board[x][y] == 1:        
        return True
    # 땅일 경우
    return False


def bfs(x, y, m, n, visited, board):
    q = []
    q.append((x, y))
    
    while q:
        x, y = q.pop()
        
        # 방문 처리
        if visited[x][y]: continue
        visited[x][y] = 1
        
        # 배추면 -> 4방향으로 퍼짐
        spread(x, y, q, m, n, board)
        
        
def spread(x, y, q, m, n, board):
    d = [(1, 0), (0, 1), (-1, 0), (0, -1)]
    for i in range(4):
        nx = x + d[i][0]
        ny = y + d[i][1]
        if 0 <= nx < n and 0 <= ny < m:
            if board[nx][ny] == 1:
                q.append((nx, ny))


def solve():
    t = int(input())
    for _ in range(t):
        m, n, k = map(int, sys.stdin.readline().split())
        board = [[0] * m for _ in range(n)]
        visited = [[0] * m for _ in range(n)]
        # 배추 위치 표시
        for _ in range(k):
            a, b = map(int, sys.stdin.readline().split())
            board[b][a] = 1
        cabbage(m, n, board, visited)

solve()
