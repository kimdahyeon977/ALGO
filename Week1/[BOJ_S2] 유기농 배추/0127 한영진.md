```
* DFS, BFS 로 가능한데, BFS로 풀었다.
    * BFS 풀이를 잊어버려서 약간 참고하면서 풀었다.
    * 이참에 BFS 로직 확실히 익히기
    * DFS 풀이도 확인하자.
* BFS 문제의 정석 같은 느낌이라 푸는데 큰 고민은 하지 않았다.
    * BFS 문제를 오랜만에 풀다보니 자잘한 부분에서 매끄럽지 않아 시간이 조금 더 걸렸다.
    -> 보다 익숙해져 빠르게 풀 수 있도록 하자.
* 최초 제출하니 RecursionErorr에 걸려 깊이 제한을 상향 시켰다.
```

```python
import sys
input = sys.stdin.readline
sys.setrecursionlimit(10**7)

from collections import deque


def bfs(x, y, arr, visited):
    max_row = len(arr)
    max_col = len(arr[0])
    visited[x][y] = True
    queue = deque([(x,y)])
    while queue:
        x, y = queue.popleft()
        dx = [0, 1, 0, -1]
        dy = [1, 0, -1, 0]

        for i in range(4):
            nx = x + dx[i]
            ny = y + dy[i]
            if nx < 0 or nx >= max_row or ny < 0 or ny >= max_col or visited[nx][ny] or arr[nx][ny] == 0:
                continue
            bfs(nx, ny, arr, visited)


t = int(input())
for _ in range(t):
    m, n, k = map(int, input().split())
    arr = [[0 for col in range(m)] for row in range(n)]
    visited = [[False] * m for row in range(n)]
    cnt = 0

    for i in range(k):
        y, x = map(int, input().split())
        arr[x][y] = 1

    for i in range(n):
        for j in range(m):
            if not visited[i][j] and arr[i][j] == 1:
                bfs(i, j, arr, visited)
                cnt += 1
    print(cnt)
```
