```
<접근법>
1. a는 작은 순대로 소팅, b는 큰 순대로 소팅
2. a > b인 지점이면 그 뒤는 더 안세고 break
```
```python
'''
115276 kb / 1212ms
'''
t = int(input())

for _ in range(t):
    cnt = 0 
    
    n,m = map(int, input().split())

    a = sorted(list(map(int, input().split()))) 
    b = sorted(list(map(int, input().split())), reverse=True)
    
    for i in a:
        for j in range(len(b)):
            if i > b[j]:
                print(i, b[j])
                cnt += len(b) - j
                break
    print(cnt)
```
