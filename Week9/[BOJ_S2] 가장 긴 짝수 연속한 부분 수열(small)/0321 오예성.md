```
문제 흐름
1. 수열에서 주어진 최대 k 만큼 값을 삭제
2. 삭제된 값에서 가장 긴 짝수 연속한 부분 수열 구하기

풀이
1. 원소의 값이 크기 때문에 한번의 탐색으로 값을 정해야함 -> O(n)
2. 값을 탐색하면서 지금까지 가장 긴 값을 구하기
3. 그 과정에서 k가 0이되면 지나간 값을 빼면서 길이를 줄여 다른 최대 길이 구하기


함수화
solve()
-> run(): 수열을 탐색
--> is_odd(): 홀수인지 확인
--> odd_process(): 홀수일 경우 처리
--> even_process(): 짝수일 경우 처리
--> update_max(): max 값을 갱신
```

```python
# 37636 KB, 84 ms
import sys
from collections import deque

def makeSettings():
    n, k = map(int, sys.stdin.readline().split())
    nums = list(map(int, sys.stdin.readline().split()))
    return k, nums

def run(k, nums):
    stack = deque()
    count = k
    now = 0
    max = 0
    for num in nums:
        if is_odd(num):
            count, now = odd_process(count, now, stack)
        else:   
            now = even_process(now)
        max = update_max(now, max)
        stack.append(num)
    return max

def is_odd(n):
    if n%2 == 1:
        return True
    return False

def odd_process(count, now, stack):
    if count == 0:
        while stack:
            pop_value = stack.popleft()
            if is_odd(pop_value):
                count += 1
                break
            else:
                now -=1
    count -= 1
    return count, now

def even_process(now):
    return now+1

def update_max(now, max):
    if now > max:
        max = now
    return max

def solve():

    k, nums = makeSettings()

    max = run(k, nums)
    
    print(max)
    
solve()
```
